# Vault Reader (Under Development)

A minimal, modern, cross-platform synced e-book reader.

</div>



### Main Features

- Self Host with supabase
- Modern Minimal UI
- Cross platform sync
- Offline first
- Rich inbuilt, Note taking
- Inbuilt dictionary and web Search
